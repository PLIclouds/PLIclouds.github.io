不定积分与定积分的概念和性质

原函数、不定积分和定积分

原函数和不定积分

简单来说在定义域内，\(F'(x) = f(x)\)则\(F(x)\)就是\(f(x)\)的一个原函数，\(F(x)+C\)就是全体原函数，也被称为其不定积分。

\[ \int f(x)\mathrm{d}x = F(x) + C \]

其实我感觉这个定义很怪。我倾向于把求不定积分理解为：求一个函数，这个函数的微分等于\(f(x)\mathrm{d}x\)。也就是说，不定积分是微分的逆运算。

结合\(\mathrm{d}f(x) = f'(x)\mathrm{d}x\)，基本上不定积分的很多基本性质都很好理解。

定积分

定积分定义很长，具体理解定义貌似没什么用。这里只放出其几何意义：函数和坐标轴围成的曲面梯形的面积代数和。

根据几何意义其基本性质也十分明确。

引入变量，则有变限积分：

\[ F(x) = \int_a^xf(t)\mathrm{d}t \]

变限积分满足：

\[ F'(x) = f(x) \]

定积分里最重要的基本定理必然是牛顿-莱布尼茨公式了：

\[ \int_a^bf(x)\mathrm{d}x = F(b)-F(a) \]

函数

变量若按照一定的对应法则总有一个对应的与其对应，则称是的函数。也就是说，只要对应关系和定义域确定，函数唯一确定。

函数除了常见的初等函数，还有分段函数，反函数，复合函数，隐函数，参数方程几种形式。

函数性质

大概包括单调性，奇偶性，周期性，有界性。要证明就按定义证明就行了。

极限

主要是数列和函数求极限相关。

数列极限

数列极限更加简单：数列在时取值始终在常数的邻域中。

除了定义法，常见的方法还有夹逼定理（放缩）和子序列收敛定理（若数列收敛于，其任何子序列都收敛于），还可以把这个转换成函数极限来求。

函数极限

函数极限分为两种，自变量趋于无穷时的极限和趋于有限值的极限（后者也就是经典的定义，对“函数的趋于”限制，对“自变量的趋于”限制），同时这两种极限也都分为左极限和右极限（如果把无穷看作一个点，其左侧就是正无穷，右侧就是负无穷，这个思想一直都很好用）。具体定义大体为：函数在x趋于一个值（包括有限和无限，后续不再赘述）时取值始终在常数的邻域中。

函数极限存在的充要条件是，左极限和右极限同时存在且相等。

无穷小和无穷大

在趋于一个值时极限为0，称其为时的无穷小量。无穷大量就是趋于无穷（无论正负）。

Positional Encoding

就是把位置信息加入词向量中，毕竟没有中间状态可以用了嘛。原文给了个三角形式的位置编码还没解释清楚，这有篇文章讲挺明白，but先观察这个函数形式：

\[ PE_{(pos, 2i)} = sin(\frac{pos}{10000^{2i/d_{model}}}) \]

\[ PE_{(pos, 2i+1)} = cos(\frac{pos}{10000^{2i/d_{model}}}) \]

这里的pos一旦大一点，例如\(pos=100，2i/d_{model} = 0.5\)和$ pos=728，2i/d_{model} = 0.5$，这两个值就特别接近，很容易让机器误解。其实还有很多这样的情况。具体的优化可以看2.2.1。

Self-Attention自注意力机制

原始的Attention模型使用了循环卷积中的隐藏状态作为注意力参数的计算方法。Transformer这里用的是另一种叫做Self-Attention的方法。其核心解决的问题就一个：
